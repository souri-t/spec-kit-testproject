# Quickstart: 指定プロンプトを毎日定時実行しWeb表示

## 準備
1. Python環境を用意（3.11以上推奨）
2. 必要パッケージをインストール
   - streamlit
   - fastapi
   - openrouter (APIクライアント)

## 設定
1. Web画面でプロンプト・実行時刻・APIキー等を登録
2. 設定はJSONファイルとして保存

## 実行
1. FastAPIサーバーを起動
2. Streamlitフロントエンドを起動
3. 指定時刻にAI問い合わせが自動実行され、結果がJSON保存・Web表示される

## テスト
- 設定未登録時はエラー表示されること
- AI問い合わせ失敗時はリトライ後エラー表示されること
- 結果保存失敗時はエラー表示されること
- Web画面で最新結果が表示されること
